# Lessons

- Stop re-submitting a vetoed proposal unchanged: UNH, JPM, and XOM were each proposed twice and vetoed twice within hours — either fix the parameter the risk agent flagged or drop the name, don't burn cycles re-pitching identical size/thesis.
- The only approved trade (AAPL pullback-long) was the sole proposal that matched the pre-vetted premarket watchlist; the six vetoed ideas (XOM/UNH/JPM) were off-list improvisations — anchor proposals to names researched premarket rather than chasing intraday breakouts.
- All three vetoed breakout longs closed green same-day (XOM +2.3%, UNH +3.5%, JPM +1.9%) in a calm sideways tape (VIX 17, RV 12%) — if the veto rationale was generic regime caution rather than name-specific, that filter is too tight for low-vol trend-continuation and should be loosened.
